# Getting started
    Prerequisite
    - mongodb 5.0.3
    - node js 14.17.1
    How to run
    > Goto the FrontEnd Folder and type
    ``` 
        npm i && cd server && npm i && npm run start && cd..
    ```
    > For windows Open up GitBash and type
    ``` 
        npm i && cd server && npm i && npm run start && cd..
    ```
    > The app will be at http://127.0.0.1:3000
